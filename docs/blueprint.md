# **App Name**: RP Coach-Up

## Core Features:

- Responsive Public Dashboard: A prominent public dashboard accessible on both mobile and desktop, showcasing general information about the platform, available programs, and calls-to-action.
- User Authentication (Student/Teacher): Secure registration and login functionality for students and teachers, allowing them to create and manage their accounts. This interacts with the specified SQL database.
- Student Interest Submission: A dedicated form enabling students to submit their specific interests for tuition classes, which is stored in the SQL database for review.
- Teacher Interest Submission: A dedicated form for teachers to submit their teaching interests and availability for tuition classes, which is stored in the SQL database.
- AI-Powered Course Match Tool: A generative AI tool that assists administrators by suggesting potential course-to-student or teacher-to-student matches based on the interests submitted by both parties.
- User Feedback System: A system allowing registered students and teachers to provide feedback on courses or the platform directly, with submissions stored in the SQL database.
- Admin Management Portal: A comprehensive admin dashboard to view, approve, and manage all submitted interest forms and to perform CRUD operations (Create, Read, Update, Delete) on registered student and teacher accounts in the SQL database.
- Java Backend Services: All server-side logic, API endpoints, and database interactions will be implemented using Java to ensure robust and scalable backend operations.

## Style Guidelines:

- The overall color scheme is light, conveying professionalism and clarity.
- Primary color: A confident and trustworthy deep sky blue (#266EDB). This hue (HSL: 220, 70%, 50%) is chosen to evoke intelligence and reliability, fitting for an educational platform.
- Background color: A serene and understated light blue (#ECF2F7). This highly desaturated shade of the primary color (HSL: 220, 20%, 95%) ensures readability and a calm user experience.
- Accent color: A vibrant cyan (#34CEF6). Positioned approximately 30 degrees counter-clockwise on the color wheel from the primary (HSL: 190, 80%, 60%), this bright and energetic hue provides a clear highlight for interactive elements and call-to-actions, ensuring good contrast and visual pop.
- Headlines and prominent titles will use 'Space Grotesk' (sans-serif) for a modern, tech-informed, and clear presentation.
- Body text and longer content blocks will use 'Inter' (sans-serif), chosen for its readability, neutrality, and professional appearance, complementing the modern feel of 'Space Grotesk'.
- Employ a set of clear, concise, and modern icons that are easily recognizable and aid navigation, particularly for educational topics and user actions.
- A responsive, adaptive layout that provides an optimal viewing experience across both mobile and desktop devices, featuring well-organized dashboards and intuitive form structures.
- Subtle, non-distracting animations for transitions between sections, form submissions, and feedback confirmations to enhance the user experience without being overwhelming.
